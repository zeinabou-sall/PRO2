 PRO2 - Kanban Board
Application web développée avec React permettant la gestion de tâches sous forme de tableau Kanban.

 Fonctionnalités
- Création de tâches
- Déplacement (drag & drop)
- Organisation par colonnes
- Interface interactive
  Technologies utilisées
- React.js
- JavaScript
- CSS
- Node.js (backend)
 Installation
  bash
npm install
npm run dev
